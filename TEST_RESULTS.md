## 下一步操作

### 1. 初始化数据库
```bash
# 创建数据库和表
mysql -u root -p < packages/database/init.sql

# 插入初始数据
mysql -u root -p equipment_management < packages/database/seed.sql
```

### 2. 配置数据库连接
编辑 `packages/backend/.env` 文件，设置正确的数据库连接信息。

### 3. 启动项目
```bash
# 在项目根目录运行
pnpm dev
```

这将同时启动前端（http://localhost:3000）和后端（http://localhost:3001）。
