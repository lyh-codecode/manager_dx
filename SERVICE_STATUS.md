## 🔧 如果遇到问题

### 服务未启动
```bash
# 检查服务状态
lsof -ti:3000  # 前端
lsof -ti:3001  # 后端

# 手动启动
cd packages/backend && pnpm dev
cd packages/frontend && pnpm dev
```

### 端口被占用
```bash
# 停止占用端口的进程
kill -9 $(lsof -ti:3000)
kill -9 $(lsof -ti:3001)
```

### 数据库连接问题
- 检查 `packages/backend/.env` 文件
- 确认数据库服务正在运行
- 测试数据库连接：`mysql -u root -p`
